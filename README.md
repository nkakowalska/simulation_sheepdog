**Spatial Simulation-Final Project:**
## **Shepherding dog**
This project is created as a final excercise for my Spatial Simulation class from the **CDE master program**.  
It includes a text file with code written in GAML to create a GAMA simulation of a shepherd dog hearding 
sheep, based on a real life video example, as well as the necessary shapefiles to create the enviroment in the simulation.
